🚀 Startup Co-Founder Finder

📌 Description
Startup Co-Founder Finder is a full-stack web application that connects entrepreneurs, developers, and startup enthusiasts. Users can create startup ideas, browse opportunities, find potential co-founders, create professional profiles, and apply to join startups. The platform simplifies collaboration for people looking to build innovative startups together.

✨ Features
🔐 User Registration & Login
🔒 Secure JWT Authentication
🚀 Create Startup Ideas
🔍 Browse & Search Startup Ideas
👥 Find Co-Founders
👤 Create & Manage User Profile
📩 Apply to Join Startups
📋 View Application Status Dashboard
📱 Responsive User Interface
💾 MySQL Database Integration

🛠️ Tech Stack
Frontend: HTML5, CSS3, JavaScript
Backend: Node.js, Express.js
Database: MySQL
Authentication: JWT (JSON Web Token)
Tools & Version Control: Git, GitHub

⚙️ Installation & Setup
1. Clone the repository
git clone https://github.com/YOUR_USERNAME/startup-cofounder-finder.git
2. Navigate to the project
cd startup-cofounder-finder
3. Install backend dependencies
cd Backend
npm install
4. Configure MySQL
Create a database named: startup_finder
Import the SQL file: Project.sql
5. Configure Database
Update Backend/config/db.js (or .env if using environment variables) with your MySQL credentials.
6. Start the backend server: npm start
Server runs at: http://localhost:5000
7. Open the Frontend
Open frontend/index.html in your browser (or serve the frontend using a local server).
